What is DS-OS?
[2014-10-5 Edit: This site, originally written in 1999, has been resurrected; it's shown here along with data files essentially unchanged.]

DS-OS is a free operating system with open source written entirely in 8086 assembly using the A86 assembler. Note that it is not compatible with any other OS, it was written and designed from the ground up.
What makes DS-OS unique?
DS-OS is a great way to learn operating systems programming. It is a full blown operating system, yet it has just the right amount of functionality and will not overwhelm beginners

DS-OS is small(about 6k total), so you don't have to sort through 1000's of lines of source code to find what you want. The source code is also heavily commented, written entirely in 8086 assembly. It was coded in A86, a very good 8086 assembler, so that anyone can make any changes to it and re-assemble it. If there is interest I can put this on GitHub.
What can I do with DS-OS?
DS-OS comes with a new assembler [2014 Edit: was never finished] called DS-ASM created especially for DS-OS. It also comes with a text editor [2014 Edit: I can give this to you if you are interested], so you can develope your own assembly programs for it. You can use any of the DS-OS code in your operating system if you would like(for free), but I would appreciate if you would let me know if you do so I can download your OS and see it! :)
List of features
Here is a list of features.
  • Makes use of the DS-FAT 16.4 file system.
  • Comes with a text editor created specifically for this OS. [2014 Edit: available upon demand]
  • Comes with a PS/2 mouse driver to add mouse support.
  • Works on floppy disks of any size if sector size is 512
  • Supports logical drives on hard disks(up to 4), similar to MS-DOS's.
  • Has been tested on and works on 8088 computers.
  • Written in 100% assembly: very fast and only 6k total!
  • Built in disk caching system for speed in file access
  • Supports file handles for easy access to files(on any drive)
  • Easy interface for applications. Includes built in ASCII <--> BINARY converters!
  • MS-DOS like command interpreter(easy to use).
  • Supports Memory Control Blocks, PSP's, deeply nested programs, and memory management.
