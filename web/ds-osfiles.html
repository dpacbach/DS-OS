Here are the DS-OS files. To install the OS onto a floppy disk, just download the
install program and the image (the first two files on the list below).

If you try to download the install program and you get a bunch of wierd
characters, try it again but this time hold down shift when you click on it.

Once you get the OS installed and have run it, type "help" at the command
prompt to get a list of commands.

Note: the image file contains a program called sin.pgrm. The
computer must have a math coprocessor installed to run this program!

File Description
install.com install program
image.flp floppy image of DS-OS
boot.a86 boot sector
cmd.a86 command interpreter
os.a86 kernel
macros.inc some macros
dump.a86 sample DS-OS application
